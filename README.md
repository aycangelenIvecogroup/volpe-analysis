Profit Intelligence Platform
Transforms raw sales data into clear, actionable business insights

 Overview
This project is an end-to-end sales performance and profitability intelligence platform built with Streamlit.
It enables business users to:

Monitor performance vs targets
Identify risks and opportunities
Understand root causes down to product (PN) level
Analyze pricing, cost and margin dynamics

 Key Capabilities
 1. Dashboard

Customer performance vs Budget / Forecast / Last Year
RunRate projection (YTD → full year)
KPI tracking (TN, Margin, Growth)
Customer segmentation (Star / At Risk / Mixed)


2. Insights Engine

Automatically identifies:

Best performers
Worst performers


Ranks customers based on margin deviation


3. Problem Detection

Highlights customers with:

Cost increase (Cost ↑)
Pricing pressure (Price ↓)


Automatically classifies risk level


 4. Customer & Product Analysis

Drill-down from:

Customer → Product → PN level


Helps identify which products drive performance


 5. Unit Economics Analysis

Full decomposition of price:

Unit Price = COGS + VCE + Margin + Variance


Explains margin changes using:

Price
Cost
Structural effects



 6. Scenario Comparison

Compare multiple scenarios (ACT vs BDG vs FCS1)
Automatically calculates:

Δ = A - B  
%Δ = (A - B) / B


Identifies performance gaps across customers/products


7. Unit Bridge (Advanced)


Breaks total performance into drivers:

Volume
Cost
Pricing
Variance



Identifies main driver automatically



 How It Works
The system applies a structured logic:

Aggregate data (customer / product / PN)
Convert into unit-level metrics
Compare scenarios
Calculate deltas
Detect patterns
Generate insights automatically

 No manual analysis required

📈Business Value

Faster decision-making
Clear visibility of performance drivers
Immediate identification of risks
Reduced manual reporting effort
Full explainability (no black-box logic)
